Здравствуйте. Меня зовут Окрут Диана. В рамках этого практического задания передо мной стояла задача разработать приложение для тестирования.
Вложенные в архив файлы содержат в себе скрипт на JavaScript(script.js), содержащий в себе логику всего приложения, и HTML документ (index.html), с помощью которого можно увидеть интерфейс программы
и взаимодействовать с ней.

Функционал приложения:
- приложение позволяет пользователю создавать собственные вопросы с вариантами ответов и указывать правильные варианты ответа;
- пользователь имеет возможность инициализировать начало тестирования и узнавать свои результаты после прохождения теста;
